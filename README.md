## Developement

Requirements:
- ruby
- gem bundler


Install dependencies:

```shell
bundle install
```

Start jekyll:

```shell
bundle exec jekyll serve --baseurl ""
```

Open your browser at [http://127.0.0.1:4000](http://127.0.0.1:4000).
